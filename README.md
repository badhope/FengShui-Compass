## 🛠️ 开坛布阵（部署使用）
### 前置条件
无需服务器、无需数据库，纯前端静态页面，凡有浏览器之处，皆可开坛！

1. 「请神」：将全部文件下载至本地，保持目录结构完整
```

│   └── theme-utils.js  【玄术核心】模式切换工具
├── pages/
│   ├── compass.html        【罗盘殿】玄空罗盘测算
```

2. 「启阵」：双击 `index.html`，即可开启玄易之门
